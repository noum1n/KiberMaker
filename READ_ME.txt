# Документация программы "Коллаж"

Данная документация описывает основные функции и использование программы "Коллаж". Программа предназначена для создания коллажей изображений, позволяя пользователю объединять несколько изображений в одну композицию.

## Установка

Программа написана на языке Python с использованием библиотеки Tkinter для создания графического интерфейса пользователя. Для запуска программы необходимо установить Python версии 3 и следующие зависимости:

- Tkinter: для создания графического интерфейса
- Pillow: для обработки изображений

Установка зависимостей может быть выполнена с помощью менеджера пакетов pip следующим образом:

```shell
pip install tkinter pillow
```

## Запуск программы

Для запуска программы выполните следующую команду в командной строке:

```shell
python collage.py
```

После запуска откроется графическое окно программы "Коллаж".

## Использование

1. Загрузка изображений:
   - Нажмите кнопку "Загрузить".
   - В появившемся диалоговом окне выберите одно или несколько изображений форматов JPEG, JPG или PNG.
   - Нажмите кнопку "Открыть", чтобы загрузить выбранные изображения.
   - Загруженные изображения будут отображаться в списке загруженных изображений.

2. Генерация коллажа:
   - Настройте количество столбцов и размер границы между изображениями с помощью слайдеров.
   - Нажмите кнопку "Сгенерировать", чтобы создать коллаж из загруженных изображений.
   - Созданный коллаж будет отображаться в окне программы.

3. Сохранение коллажа:
   - Нажмите кнопку "Сохранить", чтобы сохранить текущий коллаж.
   - В появившемся диалоговом окне выберите путь и имя файла для сохранения коллажа.
   - Нажмите кнопку "Сохранить", чтобы сохранить коллаж в выбранном месте.

4. Завершение работы программы:
   - Нажмите кнопку "x" в верхнем правом углу окна или закройте окно программы.

## Ограничения

- Поддерживаемые форматы изображений: JPEG, JPG, PNG.
- Рекомендуется использовать изображения с одинаковыми пропорциями для лучшего визуального эффекта.
- Размер границы между изображениями и количество столбцов ограничены заданными значениями.
